# 问卷第二步 - 范围说明

> **原则**：整体共 3 步。第一步为基础闭环；**第二步**在第一步之上，完成逻辑（跳题、显隐、选项显隐、选项引用、必填、多条件）、题型扩展（下拉、矩阵、排序、日期、时间、段落、分页符）、分页与限时、答卷管理与分类/交叉分析，形成「可分支、可分页、可管答卷、可分组与交叉统计」的完整问卷能力。问卷星能力对标与功能缺失出处见 **《问卷星对标-功能缺失分析》**（docs/问卷星对标-功能缺失分析.md）。

**第二步目标**：在第一步能力之上，本步完成：**逻辑**（跳题、题目显隐、选项关联显隐、选项引用、必填逻辑、多条件与/或/非）；**题型**（下拉单选/多选、矩阵单选/多选/填空、排序题、日期题、时间题、段落说明、分页符）；**问卷设置**（分页：分页符/每 N 题、单次填写限时、最大回收数）；**答卷管理**（标记无效、编辑答卷与修改记录、按答案筛选、导出仅有效/当前筛选）；**统计**（分类统计、交叉分析）。复制问卷时含题目、设置、逻辑与上述题型。**不包含**反向关联、随机抽题、循环题、公式与评分、条件嵌套、配额规则、比重/上传/表格/投票等题型、外观主题、从模板/文本创建、题库、操作日志与审计、考试与高级分析等（见第三步）。

## 一、与第一步的关系（沿用与新增）

| 维度 | 第一步 | 第二步 |
|------|--------|--------|
| 问卷基础与状态 | 四态、发布/暂停/开启/结束/复制/删除 | **沿用** |
| 题型 | 单选、多选、填空、量表（1–N） | **+ 下拉、矩阵、排序、日期、时间、段落、分页符** |
| 逻辑 | 无 | **跳题、题目显隐、选项显隐、选项引用、必填逻辑、多条件** |
| 问卷设置 | 防重复/匿名/时间/感谢语 | **+ 分页（分页符/每N题）、单次限时、最大回收数** |
| 编辑页 | 题目列表+编辑 | **+ 逻辑配置（跳题/显隐/选项显隐/选项引用/必填条件/多条件）、矩阵行列、新题型编辑** |
| 填写 | 按序填写 | **按跳题与显隐展示；分页（上一页/下一页）；限时倒计时；回收满提示** |
| 答卷与统计 | 列表、单题统计、导出 Excel | **+ 标记无效、编辑答卷、修改记录、按答案筛选；分类统计、交叉分析；导出仅有效/当前筛选** |
| 复制问卷 | 含题目与设置，不复制答卷 | **含题目、设置、逻辑与全部题型**，不复制答卷 |

## 二、逻辑设置（对应功能规格 三、3.5）

### 2.1 跳题逻辑

| 功能点 | 规格出处 | 第二步实现要求 |
|--------|----------|----------------|
| 按选项跳转 | 3.5 跳题逻辑 | **单选题、下拉单选题**：每选项可配置「选择后跳转到第 N 题」或「选择后结束答卷」；未配置则顺序下一题 |
| 无条件跳转 | 3.5 | 可选：从当前题直接跳转到某题或结束答卷 |
| 适用 | 3.5 | 先支持单选、下拉单选；多选跳题规则可约定（如选中某选项即跳） |

**实现注意**：跳转目标为题号或题目 ID；填写端按已选答案计算下一题；被跳过题目不展示，答卷中无该题答案，统计/导出留空或标注「未作答（已跳过）」。（对标问卷星：**跳题从前往后设置**，**题目关联从后往前设置**；多选不推荐跳题，推荐用题目关联。）

### 2.2 题目关联逻辑（显示/隐藏）

| 功能点 | 规格出处 | 第二步实现要求 |
|--------|----------|----------------|
| 题目显隐 | 3.5 题目关联逻辑 | 某题**显示与否**依赖前面某题**选项是否被选中**；可配置默认显示/默认不显示；支持**批量设置**相同关联条件至多题（对标问卷星） |
| 条件 | 3.5 | 单条件或多条件组合（与/或/非），见 2.6 |

**实现注意**：被隐藏题不展示，答卷无该题答案，统计/导出同跳题处理。

### 2.3 选项关联逻辑（同一题内选项显隐）

| 功能点 | 规格出处 | 第二步实现要求 |
|--------|----------|----------------|
| 选项显隐 | 3.5 选项关联逻辑 | 同一题内**部分选项的显示/隐藏**依赖前面某题选项（如性别选「男」才显示「足球/篮球」） |
| 适用题型 | 3.5 | 触发题：单选、下拉单选；被控题：单选、多选、下拉等 |

### 2.4 选项引用逻辑

| 功能点 | 规格出处 | 第二步实现要求 |
|--------|----------|----------------|
| 选项来源于前题 | 3.5 选项引用逻辑 | 本题**选项列表**来源于前面某题的**被选选项**（如「知道的品牌」多选 →「正在使用的品牌」选项仅为前面所选） |
| 适用 | 3.5 | 前题：多选、多选下拉等；本题：单选、多选、下拉等 |

### 2.5 必填逻辑

| 功能点 | 规格出处 | 第二步实现要求 |
|--------|----------|----------------|
| 条件必填 | 3.5 必填逻辑 | 某题是否必填依赖前面题目答案（如选「其他」时说明框才必填） |
| 配置 | 3.3、3.5 | 每题可配置「当某题满足某条件时本题必填」；提交校验按当前答案判断 |

### 2.6 多条件组合

| 功能点 | 规格出处 | 第二步实现要求 |
|--------|----------|----------------|
| 与/或/非 | 3.5 | 跳题、题目显隐、选项显隐、必填条件支持**多条件组合**：与(&&)、或(\|\|)、非(!) |
| 范围 | 3.5 | 条件来自「当前题之前」的题目；条件数 2～N 可配置 |

**编辑页逻辑入口**：提供**检查全部逻辑**入口，集中展示已配置的跳题、题目显隐、选项显隐、选项引用、条件必填等（对标问卷星）。**管道引用**（题干中引用前题选项文本 [q1]）：**仅单选题、多选题、单项填空题**可被后续题目引用（对标问卷星）。

**第二步不做（逻辑）**：反向关联、随机抽题、循环题、变量与公式、评分逻辑、条件嵌套 → 第三步。

## 三、题型新增（对应功能规格 三、3.2.1、3.4）

**第一步已有题型在本步可增强**（对标问卷星）：**单选题**可支持**列表单选**、**单选+填空**（选项后可跟填空，第一步已有「其他」+填空）、**单选题（图片选项）**（选项为图片）；**填空题**可支持**多项填空**且**每空单独配置校验类型与长度**；题目**选项说明**可嵌入**网址内容**（https），点击后弹出。

| 题型 | 规格出处 | 第二步实现要求 |
|------|----------|----------------|
| 单选下拉 / 多选下拉 | 3.2.1 | 下拉框单选/多选（**单级**）；选项、其他+填空；参与跳题与显隐。**说明**：问卷星仅有「多级下拉框题」，不存在普通单级下拉题；本项属本系统扩展，可选做。 |
| 矩阵单选题 | 3.2.1、3.4 | 行×列，每行选一列；行/列可编辑；每行必选/可选可配置；**必做**。矩阵**量表题**可针对选项设置分值，统计**行平均分**（公式：Σ 选项分数×被选次数 / 有效答卷份数）；**长表时隔行显示列标题**（如超 15 行时隔 10 行，手机端超 6 行隔 4 行）（对标问卷星） |
| 矩阵多选 / 矩阵填空 | 3.2.1、3.4 | 行×列多选、行×列每格一空；**可选** |
| 排序题 | 3.2.1、3.4 | 拖拽或点选排序；统计为各选项平均排名或排名分布；**记分/综合排名规则**：权值由排列位置决定（如第一位置权值 N、第二 N-1…），选项平均综合得分 = Σ 频数×权值 / 本题填写人次（对标问卷星） |
| 日期题 / 时间题 | 3.2.1、3.4 | 选择日期/时间，可限制范围；统计为分布 |
| 段落说明 | 3.2.1 | 仅展示文字，不收集答案 |
| 分页符 | 3.2.1 | 仅用于分页，不收集答案；编辑时插入，填写时按页切分 |

**第二步不做（题型）**：比重、上传文件、表格题、投票题、多级下拉等 → 第三步。

## 四、问卷级设置新增（对应功能规格 三、3.6；四、4.5）

| 功能点 | 规格出处 | 第二步实现要求 |
|--------|----------|----------------|
| 分页 | 3.6 分页 | **不分页** / **按分页符** / **每 N 题一页**（含**一题一页**）；是否允许「上一页」返回可配置；支持**批量删除分页**（对标问卷星） |
| 单次填写限时 | 3.6、4.5.1 | 如 10 分钟内提交；超时禁止或提示；填写页可展示倒计时 |
| 最大回收数 | 3.6、4.5.1 | 达到后关闭填写或提示「回收已满」；列表/统计可展示「已回收/上限」 |

**第二步不做**：配额规则（按选项/条件分桶）、每日配额、答题时间控制、问卷密码等 → 第三步。

## 五、答卷管理（对应功能规格 五、5.1）

| 功能点 | 规格出处 | 第二步实现要求 |
|--------|----------|----------------|
| 标记无效 | 5.1 | 单条/批量**标记无效**（及取消）；无效**不计入**单题统计、分类统计、交叉分析；列表筛选「仅有效/仅无效/全部」 |
| 编辑答卷 | 5.1 | 创建者可**编辑某条答卷**（修改某题答案）；代录纠错、补录用 |
| 修改记录 | 5.1 | 编辑后**保留修改记录**（谁、何时、哪题、原值/新值），便于审计 |
| 按答案筛选 | 5.1 | 答卷列表**按题目答案筛选**（如「第 2 题选了 A 的答卷」） |
| 导出范围与格式 | 5.6 | 导出可选「全部答卷」「仅有效答卷」「当前筛选结果」；Excel 可选**按选项序号** / **按选项文本** / **按选项分数**（导出值为题目/选项预设分数）；**SPSS(.sav)** 可选；身份证等长数字以文本存储（对标问卷星） |

## 六、分类统计与交叉分析（对应功能规格 五、5.2、5.4）

| 功能点 | 规格出处 | 第二步实现要求 |
|--------|----------|----------------|
| 分类统计 | 5.2 | 以某题**选项为分类维度**，得到每一类下的**单题统计**（如按性别看各题统计）；先选分类依据题，再切换查看各类下各题 |
| 交叉分析 | 5.4 | 选**两题 A、B**：以 A 选项为行（或列），展示 B 在各 A 选项下的统计；**交叉表** + 可选图表；注明分母与占比口径 |

**第二步不做**：多维度组合分类、按地区/来源分类、自定义数据源、多题交叉等 → 第三步。

## 七、编辑页、填写页与复制（第二步补充）

| 功能点 | 第二步要求 |
|--------|------------|
| 题目编辑区 | 逻辑配置：跳题、题目显隐、选项显隐、选项引用、必填条件、多条件；**检查全部逻辑**入口；矩阵题行/列编辑；新题型（排序、日期、时间、段落、分页符）编辑；选项说明嵌网址、单选题图片选项等（见三） |
| 题目列表 | 可展示逻辑标识（如「第 2 题→跳转」「第 3 题←第 1 题选 A 显示」） |

| 填写页 | 按跳题与显隐展示；**分页**（上一页/下一页）；**限时倒计时**（若开启）；**回收已满**提示；必填校验仅针对当前可见题 |
| 复制问卷 | 含题目、设置、**全部逻辑**（跳题/显隐/选项显隐/选项引用/必填/多条件）、**全部题型**（含矩阵、下拉、排序、日期、时间、段落、分页符）、分页与限时/回收数设置；不复制答卷 |

## 八、与功能规格其他章节的对应关系

- **三、问卷编辑**：本步覆盖 3.5 跳题、题目关联、选项关联、选项引用、必填逻辑、多条件子集；3.2.1 下拉、矩阵、排序、日期、时间、段落、分页符；3.6 分页、单次限时、最大回收数。3.5 反向/随机/循环/公式/评分/嵌套，3.7 外观、3.8 题库、3.9 从模板/文本创建 → 第三步。

- **六、模板与复用**：复制问卷含本步全部；从模板创建、从文本导入、题库 → 第三步。
