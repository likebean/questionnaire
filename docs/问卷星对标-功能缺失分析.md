# 问卷星对标 · 功能缺失分析

> 本文为 **《问卷星对标-功能缺失分析》** 与 **《基于帮助中心-全部正文与图片-功能缺失分析》** 的合并版，为唯一汇总文档。  
> **数据来源**：以 **docs/wjx-help** 帮助中心全部文档与图片为 base（605 页，依据 pages-index.json）；逐条对照的完整依据为 **《帮助中心-全部正文与图片》**（`docs/wjx-help/帮助中心-全部正文与图片.md`，由 help-fulltext-index.json 按一、二级菜单生成，每页含全部标题、段落与配图）。

> **目标**：列出项目文档中未覆盖或描述不足的问卷星能力，注明出处，便于补齐、对标问卷星。

**本报告结构**：第一节为数据来源与对照方式；第二～九节为**按模块的逐条缺失**（以《帮助中心-全部正文与图片》为据，每条带出处文件名）；第十节为**建议优先补充到项目文档的条目**；第十一节为**使用说明**（如何回查原文与配图）；第十二～十四节为其他模块速览、文档与帮助中心对照索引、按 helpid 清单的系统性缺失；第十五节为小结与已完成的文档补齐。
