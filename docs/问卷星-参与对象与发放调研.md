# 问卷星「参与对象」与发放方式调研

> 基于问卷星帮助中心（wjx.cn/help）与公开文档整理，便于与本系统「有哪些人可以参与问卷」能力对标。

---

## 一、填写对象 / 参与范围：问卷星怎么做

### 1.1 默认逻辑

- **谁可以填**：拿到问卷链接（或二维码、嵌入页）的人都可以填写，不做「名单内才可填」的硬性限制。
- **发放方式**：链接、二维码、邀请邮件、邀请短信、微信/公众号、嵌入网站等。

### 1.2 「指定填写对象」的几种方式

| 方式 | 说明 | 版本/入口 |
|------|------|-----------|
| **联系人 + 定向发放** | 先有「联系人」名单，再通过短信/邮件/公众号只给这批人发问卷链接；可看已参与/未参与进度 | 企业标准版（个人外部联系人 1 万）、尊享版（企业外部联系人 5 万） |
| **每人唯一链接（短信邀请）** | 针对每个手机号生成**唯一填写链接**，只有收到短信的人用自己链接可填；可查看谁填了谁没填 | 回收答卷 → 邀请短信（手动输入或 Excel 批量导入手机号） |
| **只允许从自定义来源链接访问** | 勾选后，**仅**通过「自定义来源」生成的链接可访问问卷，普通问卷链接无效 | 发送问卷 → 链接与二维码 → 自定义来源与调研员 |
| **考试 / 用户体系** | 先导入答题者名单，答题者通过指定链接或微信/短信进入；适合「名单内才能作答」 | 考试答题系统 / 用户体系 |

---

## 二、联系人管理（企业版）

- **入口**：尊享版从「通讯录」进企业外部联系人；标准版从「我的问卷」右上角「联系人」进个人外部联系人。
- **联系人从哪来**：手动添加、从问卷答卷同步、从公众号粉丝同步、Excel 批量导入、将答题者存为联系人。
- **标识**：手机号/邮箱/OpenID 至少其一，用于去重与同步。
- **标签**：可打标签，发放时按「全部 / 按标签 / 按条件」筛选，或指定手机号/邮箱/Openid 发放。
- **发放流程**：发送问卷 → 外部联系人发放 → 新建任务 → 选渠道（短信/邮件/公众号）→ 选范围（全部/按条件/指定一批）→ 发送；可查看「已参与」「未参与」进度。

---

## 三、发放渠道与「谁可以参与」的关系

| 渠道 | 是否指定对象 | 说明 |
|------|--------------|------|
| 问卷链接 / 二维码 | 否 | 谁拿到谁填 |
| 邀请短信 | 是 | 手动或 Excel 导入手机号；可为每号码生成唯一链接，实现「指定人 + 唯一链接」 |
| 邀请邮件 | 是 | 指定邮箱列表发送 |
| 微信 / 公众号邀请 | 是 | 对粉丝或指定用户发；公众号有每人每月条数等限制 |
| 外部联系人发放 | 是 | 对联系人库按范围/标签/指定名单发短信/邮件/公众号 |
| 自定义来源链接 | 可限制入口 | 勾选「只允许从自定义来源链接访问」后，仅持该来源链接的人能打开问卷 |
| 嵌入网站 | 否 | 访问该页面的人可填 |

## 四、和本系统的对照（简要）

| 能力 | 问卷星 | 本系统（第一步） |
|------|--------|------------------|
| 谁可以填 | 默认任何人持链接可填；要「指定人」则用联系人+定向发放 或 每人唯一链接 或 自定义来源限制 | 任何人持链接可填；无联系人/名单/唯一链接 |
| 参与名单 / 样本 | 有（联系人管理、标签、已参与/未参与） | 无 |
| 每人唯一链接 | 有（短信邀请按手机号生成） | 第三步可选（4.1） |
| 只允许某渠道链接可填 | 有（只允许从自定义来源链接访问） | 未做 |
| 按院系/身份限制 | 企业微信/钉钉等可见范围 | 可沿用本校院系+权限做「按院系可见」 |

---

## 五、结论与可借鉴点

1. **问卷星没有单独的「参与范围」配置页**，而是通过「联系人 + 发放方式 + 链接类型」组合实现：
   - 要**指定谁可填**：先有联系人/名单，再只给这些人发链接；或用每人唯一链接（短信邀请）。
   - 要**禁止随便扩散**：用「只允许从自定义来源链接访问」或每人唯一链接。

2. **本系统若做「有哪些人可以参与」**，可参考：
   - **轻量**：问卷设置里增加「只允许从指定链接/来源访问」（类似自定义来源），不建名单也能收窄入口。
   - **完整**：第三步的「样本管理（名单、状态、完成率）」+「每人唯一填写链接」+ 短信/邮件邀请（可选），与问卷星「联系人 + 定向发放 + 唯一链接」对齐。

3. **高校场景**：本校已有院系/用户，可做「按院系可选参与范围」（如仅某院系可填），与问卷星的「企业可见范围」类似，实现方式按本校权限模型即可。
